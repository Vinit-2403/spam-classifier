ℹ️ About This Project

📌 Project Overview

End-to-end Email & SMS Spam Classification system built using Machine Learning. The model uses TF-IDF vectorization combined with a Multinomial Naive Bayes classifier to detect spam messages with high accuracy.

🧠 ML Pipeline

Text preprocessing → TF-IDF (bigrams, 5000 features) → Multinomial Naive Bayes with Laplace smoothing. The pipeline handles URL detection, phone number normalization, and case normalization.

🛠️ Tech Stack

Python 3.10 Flask scikit-learn TF-IDF Naive Bayes NumPy Pandas Gunicorn Heroku

🚀 Deployment

Deployed on Heroku using Gunicorn WSGI server. The model is serialized with pickle and loaded at startup. CI/CD via GitHub Actions with automatic deployment on push to main.
